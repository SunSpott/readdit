### Sun Readdit Project
This is my Readdit project that uses Reddit API to build. The user can read the Reddit forum but can't write. This app feature includes searching, adding their favorites category to dive on,  and reading the Reddit forum in the live feed!

## Techniology

- HTML
- CSS
- JavaScript
- React
- Redux
- React-Redux
- Git

## Feature
- Search bar
- Favorite community which you can add or remove in navigation bar.
- Select bar with the Best, Hot, New, and Top sections on the initial page.
- See the comment under the posts.
- Compatible with mobile devices and smaller screens
